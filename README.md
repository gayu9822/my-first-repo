## Problem

Hospitals and NGOs often face shortages of essential goods because inventory data is scattered and stock-outs are detected too late.

## Solution

A Snowflake-native inventory monitoring dashboard that:
- Uploads daily stock data
- Monitors inventory health
- Predicts upcoming stockouts
- Visualizes data with heatmaps and charts
- Recommends reorder quantities with priority
